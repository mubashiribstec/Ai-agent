# Web research

1. Break the question into 2-4 sub-queries. Run `web_search` for each.
2. Open the most promising results with `web_fetch`; prefer primary/official sources.
3. Cross-check claims across at least two independent sources; note disagreements.
4. Synthesize a concise answer with the key findings first, then details.
5. Always end with a **Sources** list of the URLs you actually used.
